# Hiring On - Job & Internship Platform

A comprehensive platform connecting job seekers with opportunities and providing resources for skill assessment and preparation.

## ğŸš€ Features

### For Job Seekers
- **Job & Internship Search**: Browse and filter opportunities by category, location, and type.
- **Horizontal Scroll Interface**: Modern, responsive card layouts for Featured Jobs, Internships, and Mock Tests.
- **Skill-Based Mock Tests**: Take AI-powered and standard assessments (Tech, Management, General) to validate skills.
- **Profile Management**: effective profile and resume management.

### For Recruiters
- **Post Opportunities**: Create and manage job and internship listings.
- **Candidate Management**: Track applicants and view profiles.

### Technical Highlights
- **Framework**: Next.js 14+ (App Router)
- **Database**: Convex (Real-time database)
- **Styling**: Tailwind CSS & Framer Motion
- **UI Components**: Shadcn UI
- **Authentication**: Clerk / Custom Auth
